First start with OOP PHP.

Bertahun-tahun belajar PHP hanya sebatas sistem CRUD ternyata tidak membuat Mr.Woki cukup percaya diri. Banyaknya framework PHP membuat Mr.Woki binggung namun tidak putus asa. Akhirnya Mr.Woki putuskan untuk mendalami OOP langsung tanpa bantuan framework. Berbekal pengetahui OOP dari C++ dan Javascript yang duluan dipelajari, Mr.Woki ngebut abis belajar OOP di PHP. Kalau sudah belajar C++, bahasa seperti PHP akan nampak sangat mudah.

Is this late to start OOP PHP for Mr.Woki ?

Sebenarnya sejak tahun 2006 atau kurang lebih setahun sejak belajar PHP pertama kali, Mr.Woki sudah membeli buku OOP PHP pertama. Namun setelah ngorek-ngorek isi buku, Mr.Woki puyeng. Ketika ketemu error code pada buku, makin nambah puyengnya. Sumpah lebih mudah sistem CRUD database tanpa OOP ketimbang pakai PHP zaman itu.

Javascript & C++ is the guide for OOP PHP

Seperti sudah disinggung sebelumnya, pada akhirnya OOP PHP itu jadi mudah sekali dicerna setelah Mr.Woki ngoprek OOP-nya Javascript lalu C++. Padahal kalau diurutkan C++ itu bahasa yang lebih rumit dan panjang ketimbang Javascript. Kemudian OOP Javascript malah nampak lebih ganjil ketimbang OOP C++ dan PHP. Ganjil karena tidak ada isilah class dalam Javasript.
